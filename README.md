# DOM Rotoscope

A bookmarklet to visualize DOM structure with SVG overlay.

## About

"Rediscovering Rotoscope" by Masahiko Sato redefines rotoscope — a technique with over a century of history in which live-action footage is traced frame by frame to create animation — moving beyond simple outline tracing to extract the essential nature and underlying logic of movement as a new form of visual expression. Inspired by this rediscovery, I developed DOM Rotoscope, a tool that extracts and visualizes the outlines of elements from web pages.

# DOM Rotoscope

SVGオーバーレイを使用してDOM構造を可視化するブックマークレット。

## 制作の背景

佐藤雅彦氏による「ロトスコープの再発見」は、ロトスコープという100年以上の歴史をもつ、実写映像を1コマずつトレースしてアニメーション制作をする手法を再定義し、単なる輪郭線のトレースを超えた、新しい動きの本質や理（ことわり）を抽出する映像表現として捉えたものです。ロトスコープの再発見から着想を受け、ウェブページから要素の輪郭を抽出して可視化するDOM Rotoscopeを開発しました。
